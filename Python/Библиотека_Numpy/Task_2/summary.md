# Проект: Анализ численности населения стран с 1952 по 2007 годы на основе данных из файла countries.csv

## Описание
Проект нацелен на обработку и анализ данных о численности населения различных стран мира за период с 1952 по 2007 год.
Данные представлены в формате CSV, включают информацию по годам с интервалом в 5 лет и содержат численность населения стран

## Цели
1. Распаковать файл CSV и преобразовать данные в массивы для дальнейшей обработки
2. Преобразовать численность населения в миллионы
3. Преобразовать одномерный массив численности населения в двумерный
4. Вычислить общую численность населения всех стран в 2002 году
5. Вычислить прирост населения Алжира за каждый 5-летний период с 1952 по 2007 годы

## Методы
Для достижения поставленных целей были использованы следующие методы:  
1. **Распаковка и загрузка данных из CSV-файла:** используя модуль csv, данные были считаны из файла countries.csv
Для хранения данных о странах, годах и численности населения были созданы три одномерных массива: countries, years и populations, где каждый элемент соответствует строкам файла CSV
2. **Преобразование численности населения в миллионы человек:** для удобства анализа численность населения, представленная в тысячах человек, была преобразована в миллионы
3. **Преобразование одномерного массива в двумерный:**  данные по населению для каждой страны представлены за 12 временных интервалов
Чтобы удобнее было работать с данными, одномерный массив был преобразован в двумерный, где каждая строка представляет одну страну, а каждый столбец — год
4. **Вычисление общего населения всех стран в 2002 году:** был использован 10-й столбец двумерного массива, который содержит данные по 2002 году
5. **Расчет прироста населения Алжира:** был выбран соответствующий ряд из двумерного массива. Прирост населения для каждого 5-летнего интервала был рассчитан как разница между соседними значениями

## Результаты
1. Данные из файла CSV успешно преобразованы и сохранены в массивы для дальнейшей обработки
2. Численность населения преобразована в миллионы для удобства анализа
3. Одномерный массив численности населения был преобразован в двумерный, где каждая строка соответствует одной стране, а столбцы — годам
4. Общая численность населения всех стран в 2002 году составила 5,886,977,579 человек
5. Прирост населения Алжира за каждый 5-летний период с 1952 по 2007 годы был рассчитан и составил: [991331, 730092, 1759551, 2000288, 2392017, 2880949, 3221203, 3043417, 2773642, 2215127, 2046074]
